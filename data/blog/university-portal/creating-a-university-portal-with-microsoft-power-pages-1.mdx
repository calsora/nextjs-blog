## Design

When I start to a build a portal I like to draw up quick mockups to see what the portal could look like as it can spark other questions around the entity relationship diagram etc. So to do this, before I would use Figma but now I use [v0.app](https://v0.app/) that generates mockups using prompts.

I found out about this tool from [Dennis Doerflinger](https://www.youtube.com/watch?v=KwaBkC4iXzQ). To create a good design prompt I would recommend generating this with your AI model of choice, I used ChatGPT.

As you can see the design is really simple and I wanted to cover the main tables at a high-level (conceptual) level to get going.

With all portal projects, the account and contact are really important for security so student and other tables are linked to the contact table.

As you can see the design really helped me as I was able to take parts of the screen away and create tables for it, for example the schedule table that will store the modules lectures.
