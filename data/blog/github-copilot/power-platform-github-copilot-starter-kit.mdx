As a newbie to using Github Copilot, I found it wasn't that clear where to start with agents, skills and instructions and to structure a generic repo. Therefore, I've built a lightweight repo to help people getting started using Github Copilot with the Power Platform.

[Repo](https://github.com/calsora/power-platform-copilot-kit/)

It includes ....

- Agents from Power Pages to Azure.
- Instructions for best practices to Javascript web resources.
- Skills for Power Platform architecture.

### **Awesome Copilot**

Some of the components within the solution were taken from the [Awesome Copilot Repo](https://github.com/github/awesome-copilot). It's a great resource that's worth saving! It also includes a [learning hub](https://awesome-copilot.github.com/learning-hub/) that discusses the differences between skills to agents that I would really recommended too!
